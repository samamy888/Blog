# Sweetalert2 套件複習

[官方文件網址](https://sweetalert2.github.io/)

我覺得這款套件的alert框很好看

搭配icon做出的演示更是妙不可言

自從知道這個之後就不再使用原本的alert了

## CDN引入

```js
<script src="https://cdn.jsdelivr.net/npm/sweetalert2@9"></script>
```

## 語法

```js

  'Good job!', //標題 必填
  '測試成功!', //內容 非必填
  'success' //Icon 非必填
)
```
